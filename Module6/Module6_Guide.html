Models of Neighborhood Change

Module Overview

This module uses regression analysis to explore how changes in demographic variables influence changes in house prices over time. You will learn to distinguish between predictive and explanatory regression models — a foundational distinction for policy analysis.

Learning Objectives

After completing this module, you will be able to:

  1. Explain the difference between prediction (using 2000 levels of X to forecast Y) and explanation (using changes in X to explain changes in Y).
  2. Articulate why cross-sectional models are vulnerable to omitted variable bias and how change-based models partially address this.
  3. Distinguish between explanatory factors (drivers of change) and policy variables (levers that planners can actually manipulate).
  4. Run and interpret OLS regression models predicting change in median home value.
  5. Use cluster analysis to predict neighborhood change by neighborhood type.
  6. Adjust home values for inflation using a CPI multiplier.

Key Concepts: Prediction vs. Explanation

This module introduces two regression frameworks that look similar but serve very different purposes:

Model 1: Predicting Change

( Y2010 − Y2000 ) = b0 + b1 · X2000

This model uses levels of X at the start of the study period to predict subsequent changes. It is a prospector model — it only uses data available at the beginning to forecast the future. This is useful for investors or planners who want to predict which neighborhoods will appreciate, but the coefficients should not be interpreted causally because of omitted variable bias.

Model 2: Explaining Change

( Y2010 − Y2000 ) = b0 + b1 · ( X2010 − X2000 )

This model uses changes in X over the study period to explain changes in Y over the same period. By looking at changes rather than levels, we move closer to causal inference. If unemployment rates rise in a tract and home values fall, we have stronger evidence that unemployment drives home value decline, not merely that it is correlated with other factors.

Why This Matters for Policy

Identifying a variable as a driver of change does not automatically make it a policy lever. We can mandate smaller class sizes, but we cannot mandate that college graduates move to a specific city. An influx of graduates may raise home prices, but migration patterns are outside the typical planner’s control. The lab asks you to think critically about which factors are actionable.

This basic insight — that levels predict but changes explain — is the foundation of fixed effects models and difference-in-difference designs that you will encounter in more advanced coursework.
