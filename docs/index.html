Data Downloads

Each lab uses Census ACS data pulled via the tidycensus R package. If you have trouble with the Census API or need offline access, local CSV backup files are available in the repository’s data/ folder, organized by lab.

Browse Data Files on GitHub →

  • Lab 1 & 2: US counties, AZ tracts, Maricopa block group hardship data
  • Lab 3: Maricopa block groups + external overlay data (e.g., EPA TRI)
  • Lab 4: Maricopa block group hardship data (for spatial weights & LISA)
  • Lab 5: Maricopa block groups for 2015 and 2023 (temporal comparison)
  • Lab 6: Multi-scale datasets (counties, tracts, block groups, 2015 & 2023)

Note: CSV files do not include geometry. To map the data, join with shapefiles or use tigris to download boundaries.
