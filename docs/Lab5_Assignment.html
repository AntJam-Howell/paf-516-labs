Lab 5 Assignment

Temporal Change Analysis

Lab Overview

Compare multi-year ACS data (2015 vs. 2023), compute change scores, and visualize temporal shifts in economic hardship using diverging color palettes.

What You Will Do

  • Pull ACS data for two time periods (2015 and 2023)
  • Join and compute change scores between periods
  • Create diverging choropleth maps showing improvement vs. worsening
  • Analyze spatial patterns in temporal change
