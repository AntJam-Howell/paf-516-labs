Module 5 — Visualizing Neighborhood Changes

PAF 516: Community Analytics

Module Overview

This module bridges the gap between descriptive spatial analysis (Modules 2–4) and predictive modeling. You will build a baseline regression model that predicts changes in median home value (MHV) between 2000 and 2010 using census-derived neighborhood characteristics. Along the way, you will learn principles of effective cartographic visualization.

Learning Objectives

After completing this module you will be able to:

  1. Explain why median home value is a useful proxy for neighborhood quality using hedonic pricing theory.
  2. Prepare a panel dataset of census tracts by merging 2000 and 2010 variables, filtering rural tracts, and adjusting for inflation.
  3. Diagnose variable skew, multicollinearity, and non-linearity before fitting a regression model.
  4. Interpret regression coefficients and calculate standardized effect sizes to compare predictor importance.
  5. Design choropleth maps that accurately convey spatial patterns using appropriate color scales and binning strategies.
  6. Distinguish between neighborhood change (dollar amounts) and growth rates (percentages) as outcome measures.

Learning Materials

Neighborhood Change

For this project we will measure neighborhood change as a change in median home values in a community.

Home values are a reliable measure of qualitative changes to a neighborhood because the sales price of a home already takes into account all improvements to the home, as well as all improvements to the surrounding community like new parks, improved schools, and enhanced amenities like gyms and grocery stores. These are called “hedonic pricing models” in economics. Things like income levels or unemployment rates can change independently of improvements or decline in the community, so they are not great measures of neighborhood change. Home values, on the other hand, will typically increase only if demand for the neighborhood has increased, which typically occurs when the neighborhood or surrounding communities improve.
