LAB 5 — ASSIGNMENT

Temporal Change Analysis

Download the assignment file, complete the three questions, and submit your .qmd and rendered .html to Canvas.

Assignment Workflow

This file is identical to the tutorial
The pre-filled code (Steps 1–7) produces the same output as the tutorial. If this file is in the same folder as your tutorial QMD, you can knit immediately and see the same results. If you placed it in a different location, run the packages chunk first, then knit.

The Three Questions

Q1 Interpret the Direction of Change Text answer — no code required

After knitting, review the change score summary (Step 5) and the diverging change map (Step 6), then answer:

What is the overall direction of change — is hardship generally increasing or decreasing across Maricopa County CBGs? Are the patterns from Lab 4 (hot spots) worsening, improving, or stable?

Q2 Temporal Change at the Arizona Tract Level Code + output required

Rerun the temporal change analysis at the Arizona TRACT level (geography = "tract", state = "AZ") instead of Maricopa CBGs:

  • Seven scaffold chunks are provided: q2-pull-p1, q2-pull-p2, q2-build-index, q2-join, q2-summary, q2-change-map, and q2-sidebyside
  • Pull both periods (2015 and 2023), compute the change score, and map statewide
  • Show complete code and output
Q3 Compare Statewide Tracts vs. Maricopa CBGs Text answer — no code required

Compare the results from Q2 with the Maricopa CBG maps from Steps 5–7:

  • Pattern comparison: Does the statewide view reveal different patterns? Where is change most concentrated?
  • Scale comparison: What does comparing tracts (statewide) to block groups (single county) reveal about how geographic scale affects what you can see?
  • Policy relevance: When would you use each scale for policy analysis?
